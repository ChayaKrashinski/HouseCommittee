כרמלה - הוראות הטמעה

1. יש להעתיק את תייקית Fonts לתיקיית ה-CSS באתר שלכם.

2. יש להוסיף את הקוד הבא לתחילת קובץ ה-CSS:

@font-face {
font-family: 'Carmela';
src: url('Fonts/carmela.eot');
src: url('Fonts/carmela.eot?#iefix') format('embedded-opentype'),
url('Fonts/carmela.woff') format('woff'),
url('Fonts/carmela.ttf') format('truetype');
font-weight: normal;
font-style: normal;
}

3. יש להגדיר את הגופן, לדוגמא:

body {
font-family:Carmela;
font-size:15x;
line-height:18px;
font-weight:normal;
}

חשוב! נכון לכרגע לא קיימת גרסת בולד, ולכן אין להשתמש בגופן לצורך הדגשות, הדפדפן מעוות את הגופן.

כדי לוודא שהגופן מוגדר כמו שצריך, שימו לב לשמות התיקיות והקבצים, ולאותיות גדולות או קטנות.
